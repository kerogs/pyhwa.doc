## Quick installation
1. First of all, you need to install the latest version of PyHwa, available on our github.

    :::tip
    Download the latest version with this link : [Click here](https://github.com/kerogs/releases/latest)
    :::

1. When you've downloaded, you'll find a file named ``/start_server.exe``. Simply run this file, and the installation of dependencies and server startup will take place automatically.
